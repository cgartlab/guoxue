孝悌为仁之本

《论语》· 学而篇 · 第二章

"其为人也孝弟而好犯上者，鲜矣；不好犯上而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！"

论语
讲义

本章出处与背景

本章出自《论语·学而》第二章，是继开篇"学而时习之"之后，有子提出的关于道德修养根基的重要论述。

  • 本章为有子所述，有子名有若，字子有，孔子弟子
  • 有子比孔子小33岁，在弟子中以沉稳持重著称
  • 《论语》中唯有曾参与有子的言论被称为"子"（尊称）
有子曰："其为人也孝弟而好犯上者，鲜矣……"
讲义

原文逐句解读（上）

第一句：其为人也孝弟而好犯上者，鲜矣

"孝弟"：孝敬父母（孝），友爱兄长（弟，通"悌"）。
"鲜"：很少。整句意思是：那些孝敬父母、友爱兄长的人，却喜欢冒犯长辈上级的，是很少见的。

其为人也孝弟，而好犯上者，鲜矣。

第二句：不好犯上而好作乱者，未之有也

不好犯上却喜欢作乱的人，从来没有过。这句话从反面论证了孝悌与社会稳定的关系。

讲义

原文逐句解读（下）

第三句：君子务本，本立而道生

"务本"：专心致力于根本。
"道"：儒家的最高道德准则，即"仁道"。

君子务本，本立而道生。

意思是：君子致力于根本，根本建立了，道也就随之产生了。 这句话强调了基础功夫的重要性。

第四句：孝弟也者，其为仁之本与

"与"：通"欤"，语气词，表感叹。
意思是：孝敬父母、友爱兄长，这就是仁的根本吧！

讲义

核心思想：孝悌是仁的根本

有子这段话构建了一个严密的逻辑链条：

1 在家孝悌（敬父母、爱兄长）
2 自然不会犯上（尊重长辈）
3 更不会作乱（危害社会）
✓ 这就是"仁"——推己及人的爱心
孝弟 → 不犯上 → 不作乱 → 仁

有子通过一个三段论，将个人修养（孝悌）与社会责任（不犯上作乱）紧密联系起来，揭示了儒家"修身齐家治国平天下"的逻辑起点。

讲义

孝与悌：儒家的两块基石

孝

对父母的尊敬与赡养，是一切道德的起点。《说文解字》："孝，善事父母者。"

父母在，不远游，游必有方。

悌

对兄长的敬爱与顺从，是孝的延伸与扩展。"悌"从"心"从"弟"，心服于弟道。

兄道友，弟道恭。兄弟睦，孝在中。

一个人如果连生养自己的父母都不敬不爱，又怎会真心关爱他人、为社会尽责？孝悌是做人的根本，也是仁的起点。

讲义

君子务本：地基与高楼

有子用了一个极其生动的比喻——盖房子先打地基：

  • 地基不稳，房子迟早倒塌
  • 做人没有孝悌的根基，"仁道"就是空中楼阁
  • 本立了，道自然生——这是水到渠成的功夫
本立而道生

《大学》进一步发挥："身修而后家齐，家齐而后国治，国治而后天下平。"——一切从"孝悌"这个根本开始。

常见易错点解析

易错点一："弟"读 tì，不读 dì

"孝弟"的"弟"在这里读 tì（通"悌"），指友爱兄长。"兄弟"的"弟"才读 dì。两字同形但含义有别。

易错点二：谁是"有子"？

有子不是孔子的儿子，而是孔子的弟子有若。因其在弟子中德行出众，《论语》中破例以"子"尊称。

易错点三："本立而道生"的"道"指什么？

此处的"道"指儒家的仁道（最高道德准则），不是道家的"道"，也不是一般的道理、方法。

知识拓展

孝悌思想的现代意义

  • 在家庭中践行孝悌，是培养责任感和感恩之心的起点
  • 推己及人，由家庭之爱扩展到社会之爱，正是"仁"的实践路径
  • "务本"思想启示我们：做任何事都要先打好基础，不可急功近利

相关经典链接

  • 《孟子·梁惠王上》："老吾老，以及人之老；幼吾幼，以及人之幼。"——孝悌的扩展
  • 《孝经》："夫孝，德之本也，教之所由生也。"——直接继承本章思想
  • 《大学》："身修而后家齐，家齐而后国治。"——由孝悌到家国天下
